Supported files: Binary R-files from ECG Browser / Viatom Holter monitors.
File names must match the pattern R{YYYYMMDDHHMMSS} or {device_id}R{YYYYMMDDHHMMSS}.

What happens:
1. Files are decoded from delta-compressed binary format
2. ECG signal is resampled from 125 Hz → 100 Hz
3. R-peaks are detected using PhysioKit
4. Each beat is classified (Normal / PAC / PVC) using HeartKit BEAT-3-EFF-SM
5. Arrhythmia episodes are identified
6. Interactive timeline report is generated
